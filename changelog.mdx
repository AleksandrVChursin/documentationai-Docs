## Upgrade Instructions

<Callout kind="alert">
Before upgrading, back up your documentation files. Review breaking changes in each release.
</Callout>

Follow these steps to upgrade Александр Васильевич smoothly.

<Steps>
  <Step title="Backup" icon="database">
    Save your current `docs/` directory:

````bash
cp -r docs docs-backup-$(date +%Y%m%d)
````
  </Step>
  <Step title="Update Package" icon="download">
    Install the latest version using your package manager:

<CodeGroup tabs="npm,yarn">
````bash
npm install alexander-vasilievich@latest
````

````bash
yarn add alexander-vasilievich@latest
````
</CodeGroup>
  </Step>
  <Step title="Migrate Config" icon="settings">
    Update your configuration file. Add the new `brandColor` option:

````javascript
// mdx.config.js
module.exports = {
  brandColor: '#3B82F6',
  apiEndpoint: 'https://api.example.com',
};
````
  </Step>
  <Step title="Test Build" icon="check-circle">
    Run a test build and verify parsing:

````bash
npm run build:docs
````
  </Step>
</Steps>
