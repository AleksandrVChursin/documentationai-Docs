## Overview
Group and manage your docs for clarity.

<Steps>
  <Step title="Create folders">
    Use the sidebar to add new folders for topics or teams.
  </Step>
  <Step title="Add pages">
    Click "+ New Page" in any folder to start documenting.
  </Step>
  <Step title="Reorder content">
    Drag and drop folders or pages to change their order.
  </Step>
</Steps>
